# 🚗 Bluetooth Controlled Car — Day 1

## 📌 Description
Day 1: Bluetooth controlled robot car using Arduino and HC-05 (forward and stop functionality)

---

## 🎯 Objective
- Establish Bluetooth communication
- Control basic movement of robot car

---

## ✅ Features Implemented
- Forward movement
- Stop functionality
- Bluetooth communication via HC-05

---

## 🔧 Commands
- F → Forward  
- S → Stop
